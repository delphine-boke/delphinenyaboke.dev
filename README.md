## Tech Stack & Architecture

This site is built as a **modern, maintainable static portfolio**:

- **Python + Jinja2** — Content is managed in Python for easy updates
- **Tailwind CSS** — Clean, responsive, professional design
- **GitHub Pages** — Free, fast, and reliable hosting
- **Cloudflare** — Custom domain + DNS management

The architecture is intentionally simple and developer-friendly so I can maintain and extend it easily over time.

## Local Development

```bash

# clone the repository
git clone https://github.com/delphine-boke/delphinenyaboke.dev.git
cd delphinenyaboke.dev

# Install dependencies (if any)
pip install -r requirements.txt

# build the site
python build.py

# preview locally
python -m http.server 8000

```
